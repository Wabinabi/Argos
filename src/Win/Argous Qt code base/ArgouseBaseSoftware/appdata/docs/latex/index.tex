 The Argous S7 (\mbox{\hyperlink{namespace_a_s7}{AS7}}) Project is the design and implementation of an autonomous mine surveying drone for a Swinburne University Final Year Project.\hypertarget{index_autotoc_md10}{}\doxysection{Outline}\label{index_autotoc_md10}
The \mbox{\hyperlink{namespace_a_s7}{AS7}} project contains two parts\+: the on-\/board and off-\/board software The on-\/board software is loaded onto the ESP32 and is responsible for the main control and collection of data. The off-\/board software runs on Windows 10 and provides data visualisation and diagnostics for the drone.


\begin{DoxyItemize}
\item On-\/board software can be found under {\ttfamily src/esp32}
\item Off-\/board software can be found under {\ttfamily src/win10}
\item Ardu\+Pilot configuration can be found under {\ttfamily src/ardupilot}

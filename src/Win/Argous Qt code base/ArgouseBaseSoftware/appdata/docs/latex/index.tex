 The Argous Sector 7 (\mbox{\hyperlink{namespace_a_s7}{AS7}}) Project is the design and implementation of an autonomous mine surveying drone for a Swinburne University Final Year Project. The main repository can be found here\+: \href{https://github.com/Wabinabi/Argous}{\texttt{ https\+://github.\+com/\+Wabinabi/\+Argous}}\hypertarget{index_autotoc_md59}{}\doxysection{Outline}\label{index_autotoc_md59}
The project is composed of the following modules\+:


\begin{DoxyItemize}
\item The \mbox{\hyperlink{namespace_a_s7}{AS7}} Quadcopter, which itself is composed of\+:
\begin{DoxyItemize}
\item A modified Sector 7 Racing Drone
\item An accompanying electronics and sensor package mounted under the drone containing the ‘on-\/board software’, documented within the \mbox{\hyperlink{namespace_a_s7}{AS7}} Namespace
\end{DoxyItemize}
\item The \mbox{\hyperlink{namespace_a_s7}{AS7}} Base Station Software, or ‘\+Off-\/board software’
\end{DoxyItemize}

Configuration and source files can be found in the following locations\+:


\begin{DoxyItemize}
\item On-\/board software can be found under {\ttfamily src/esp32}
\item Off-\/board software can be found under {\ttfamily src/win10}
\item Ardu\+Pilot configuration can be found under {\ttfamily src/ardupilot}
\end{DoxyItemize}\hypertarget{index_autotoc_md60}{}\doxysection{Table of contents}\label{index_autotoc_md60}

The On-\/\+Board software is written within the \href{https://www.arduino.cc/en/software/}{\texttt{ Arduino IDE}} with the ESP32 Dev Kit.\hypertarget{index_autotoc_md66}{}\doxysubsubsection{Adding the ESP32 to the Arduino IDE}\label{index_autotoc_md66}
The add-\/on for the Arduino IDE can be found by adding the ESP32 Library URL to the Additional Board Manager\+:


\begin{DoxyEnumerate}
\item Head to {\ttfamily file} -\/\texorpdfstring{$>$}{>} {\ttfamily preferences} and add the following to {\itshape Additional Board Manager URLs}
\begin{DoxyEnumerate}
\item {\ttfamily \href{https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json}{\texttt{ https\+://raw.\+githubusercontent.\+com/espressif/arduino-\/esp32/gh-\/pages/package\+\_\+esp32\+\_\+index.\+json}}}
\end{DoxyEnumerate}
\item Press OK, and head to {\ttfamily Tools} -\/\texorpdfstring{$>$}{>} {\ttfamily Board\+:…} -\/\texorpdfstring{$>$}{>} {\ttfamily Boards Manager}. Searching for ESP32 should show the library package.
\item Hit install, and you should be able to see {\ttfamily ESP32 Dev Kit} under {\ttfamily Boards.}
\end{DoxyEnumerate}

A more extensive guide can be found \href{https://randomnerdtutorials.com/installing-the-esp32-board-in-arduino-ide-windows-instructions/}{\texttt{ here}}.\hypertarget{index_autotoc_md67}{}\doxysubsubsection{Required Dependencies and Libraries}\label{index_autotoc_md67}

\end{DoxyEnumerate}
